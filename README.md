## Contribute

### install adr-tools
This repo uses [ADRs](https://github.com/npryce/adr-tools) to record decisions and recommendations.

```bash
brew install adr-tools
```

### create a new recommendation
Change directory to `ic` or `team`
```bash
adr new new-recommendation
```

###  generate toc
```bash
make toc
```
